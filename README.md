# Personal Blog System

## Project URL
https://github.com/htetkaung594/Roadmaps_Training_Projects/tree/main/Personal_Blog

## Description
A full-stack web application for managing personal blog posts, built with Spring Boot, Thymeleaf, and Oracle Database.

## Features
* **Full CRUD Support**: Create, Read, Update, and Delete blog articles.
* **Data Integrity**: Optimized controller logic to prevent null overwrites.
* **Modern UI**: Card-based layout with custom date formatting (Aug 7, 2025).
* **Dynamic Sorting**: Articles are sorted in descending order (newest first).

## Technologies
* Java 17, Spring Boot 3.x
* Oracle Database, Spring Data JPA
* Thymeleaf, HTML5, CSS3
